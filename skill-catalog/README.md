# スキル図鑑

三輪さん向けのスキル整理アプリ。

## 目的

英語のフォルダー名は内部IDとして残し、日本語の短い表示名で探せるようにする。

## 方針

- フォルダー名は壊れにくい英語IDのまま残す。
- 画面では日本語短名を主役にする。
- 検索は日本語名、内部ID、発動ワードを対象にする。
- 関係マップで、読むべきスキルのつながりを見える化する。
- 追加スキルは前の内容を消さずに増やす。

## 次回からの登録ルール

- 新しいスキルやゲートを作る時は、日本語短名を必ず決める。
- 内部IDは安定した英語フォルダー名のまま残す。
- 必要ならスキル図鑑へカードを追加する。
- 同じターンで追加できない時は、未完了として報告する。
